AWS CLI S3 Bucket Commands

March 28, 2021 by Ron

One of the most basic features of AWS (Amazon Web Services) is the ability to store and create access rules for files on a service called Simple Storage Service or S3 for short. The S3 storage can be divided into buckets of your choosing so you can organize your files. You can have files for a static website, picture files, log files, tax files, invoices, bills and the list goes on.

Amazon calls files that are stored in the S3 objects, but since I come from an object oriented programming background that confuses me. We have a whole different definition for the term object that clashes with this.
