Mastering AWS

AWS CLI Scripts for S3 buckets

Command Description
aws configure Used to cofigure a user's access key id and secret acess key
aws s3 ls Used to list the s3 buckets a user has access to.
aws s3 mb s3://[bucket name] Used to create a new bucket. Special naming rules apply
aws s3 rb s3://[bucket name] --force Used to delete a bucket. Special naming rules apply
aws s3 cp [file name] s3://[bucket name] Uploads a file from a machine to an s3 bucket
aws s3 cp [folder name] s3://[bucket name] --recursive Uploads an entire folder from a machine to an s3 bucket
aws s3 cp [folder name] s3://[bucket name] --recursive --exclude * --include *.html Uploads only the *.html files from a machine to an s3 bucket
aws s3 cp [folder name] s3://[bucket name] --recursive --exclude * --include *.html --include *.png Uploads only the *.html and *.png files from a machine to a s3 bucket
aws s3 cp s3://[bucket name] [file name] Used to download a file from an s3 bucket to a machine
aws s3 cp s3://bucket name] [folder name] -- recursive Used to download the contents of a s3 folder to a machine
aws s3 cp s3://bucket name] cp[folder name] --recursive --exclude * -- include *.html Used to download only the html files from a s3 bucket to a machine
aws s3 cp s3://bucket name] [folder name] --recursive -- exclude * -- include *.html --include *.png used to download the html and png files from a s3 bucket to a machine.
aws s3 sync [file name] s3://[bucket name] Used to sync a single file between a machine and a s3 bucket
aws s3 sync [folder name] s3://[bucket name] --exclude * --include *.html --include *.png Used to sync only *.html and *.png files between an s3 bucket and a machine Deletions are not included by default
aws s3 sync [folder name] s3://[bucket name] --exclude * --include *.html --include *.png --delete Used to sync only *.html and *.png files between an s3 bucket and a machine Deletions are included
